# 달토끼 벨루나 스케줄 (v2)
- index.html: 메인 페이지
- assets/: 캐릭터 이미지
- assets/thumbs/: 썸네일(미리보기)

## 내 캐릭터로 교체
- assets/velluna_01.png 같은 이름으로 덮어쓰기
- 썸네일도 함께 교체하려면 assets/thumbs/velluna_01.png 도 덮어쓰기

## GitHub 업로드 팁(웹)
- 폴더 업로드가 막히면:
  1) 레포에서 Add file -> Create new file
  2) 파일명에 assets/thumbs/.keep 입력 후 커밋
  3) 그 다음 Upload files로 파일 업로드
