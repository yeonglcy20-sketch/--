# 달토끼 벨루나 스케줄 (v5)
## 변경사항
- 고정 MON~SUN 제거 → 원하는 요일/라벨을 직접 추가하는 방식
- 추가 시: (요일/라벨) + (시간 00:00) + (내용) → 시간 기준 자동 정렬
- 요일은 왼쪽에서 ×로 삭제 가능
- 이미지 업로드 시 위/아래 늘어남 느낌 방지: 기본 object-fit=contain + EXIF 방향 반영
